Self-hosted socks5 server in a single command on Ubuntu

Create self-hosted socks5 server in a single command on Ubuntu Server.

socks5

curl https://alextikhonov.github.io/scripts/socks/ubuntu.txt | sudo bash

If you would like to manually set port and/or password: 

export   PORT=8080;       export       PASSWORD=mypass        curl
							https://alextikhonov.github.io/scripts/socks/ubuntu.txt |  sudo --preserve-env bash

This creates self-hosted SOCKS5 server powered by Dante. Supported Linux distributions:

  • Ubuntu 16.04 Xenial
  • Ubuntu 18.04 Bionic
  • Ubuntu 20.04 Focal

P.S.

This is just a basic proxy, very simple to install and use. If you are interested in a more functional and complex soltion, you may check out Streisand Effect.
